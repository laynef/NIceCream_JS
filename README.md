## Installation

`npm install -g nice-cream` <br/>
`nice arr`<br/>

`meteor npm install` <br/>

### To add a component

`cd <project-name>`<br/>
`nice make <component-name>`<br/>

### To add a directory

`cd <project-name>`<br/>
`nice mkdir <component-name>`<br/>
